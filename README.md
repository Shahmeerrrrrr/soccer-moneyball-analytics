

# ⚽ Soccer Moneyball Analytics.

Applying **Moneyball-style data analytics** to a FIFA 23 dataset (18K+ players) to uncover hidden gems, maximize club value, and scout elite talent.

---

###  Core Engineering.

* **Growth Score:** $\text{Potential} - \text{Overall}$ *(Tracks developmental ceiling)*
* **Value Per Rating:** $\frac{\text{Value EUR}}{\text{Overall}}$ *(Measures financial efficiency)*
* **Elite Prospect:** `Age <= 23` AND `Potential >= 85` *(Filters for future superstars)*

---

### Tech Stack & SQL.

* **Tech:** PostgreSQL | Tableau | VS Code | Math
* **SQL:** Data Cleaning • Feature Engineering • CASE Statements • Window Functions • `GROUP BY / HAVING`

---

### Tableau Dashboard.

1. **KPI Summary:** High-level executive overview of the global market.
2. **Future Stars:** Dedicated scouting board for high-growth talent.
3. **Top Valuable Players:** Live shortlist of football’s most valuable assets.
4. **Club Analysis:** Deep-dive into youth academy development trends.
